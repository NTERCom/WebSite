Notes for: 6/19/2013

Several new attendees were at this meeting. Welcome to all!

It was decided that meetings should be every two weeks, for the time being. The next meeting was pushed to July 10th at 11 AM Central time due to the 4th of July holiday.

Meeting Recap and Action Items

The second meeting of the NTER Technical Users Group reviewed the topics from the kickoff meeting on May 29th.

Source Code Respository summaries and discussion. Three repositories had been reviewed for functions and features following the May 29 meeting. After a discussion of the functions and features of GitHub, SourceForge and GoogleCode, it was decided that GitHub would most likely be selected as the repository for the NTER code, and SourceForge for wikiâ€™s and support forums. Links to these sites will be posted on the NTERcom.org website, which will be used as the hub for the NTER Technical User Group.

Other discussion points included the need to identify resources for the continued development and hosting of the open source community. Items included in this are identification of ownership for code issuance, tech support, hosting, etc. In the interim, Rick Fincher will add a Liferay instance for the public-facing aspect of the open source site.

As the group continues to grow, it was noted that there will need to be sub-groups that gather to focus on their special interests: programmers, educators, system administrators, students, with a cross-over group to ensure maximum efficacy across users and developers.

The need to better understand the various NTER projects currently underway by NTERcom member groups will be addressed by uploading a project summary into the NTERcom.org website. Rick Fincher will add the text to the site, which will be supplied by each of the Project Managers for the various NTER projects (i.e. Dr. Berryman for IGEN)

Still to be determined is the date and participants for the government presentation by NTER users sometime in July.

Future meetings will take place every two weeks on Wednesdays at 11am CT, and perhaps move to monthly once the sub-groups have been developed.

Notes for: 5/29/2013

Thank you all for joining the kickoff meeting for the NTER Technical Users Group. The meeting highlighted the need for a sharing of knowledge and technical information within the NTER user community, and we are off to a solid start. Each of your contributions are valuable to the forward momentum of building and expanding an open-source community that will create a scalable and sustainable infrastructure for the NTER platform.

Several members volunteered to look into code repositories and report back. The repos discussed were Google Code, SourceForge, and GitHub.

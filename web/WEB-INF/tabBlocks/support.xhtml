Resources from the NTER Com Users Group

Nter Com has set up a SourceForge project for NTER. The project page (SourceForge NTER Project) has buttons at the top to take you to various tools. The links below will take you directly to the various resources.

We are just getting started, so check these resources often for new NTER resources.

  • Wiki

    The Wiki is where documentation is stored:
    https://sourceforge.net/p/nter/wiki/Home/

  • Discussion Forum

    The Discussion Forum is where NTER users can discuss issues among themselves related to running NTER, course authoring, and problems:
    https://sourceforge.net/p/nter/discussion/

  • Blog

    The Blog is where announcements and articles about what's new in the NTER Community will be posted:
    https://sourceforge.net/p/nter/blog/

  • Tickets

    The Tickets area is where NTER Users can file bug reports:
    https://sourceforge.net/p/nter/tickets/

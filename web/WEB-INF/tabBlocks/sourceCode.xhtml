GitHub Repositories

A GitHub "Organization" for NTER Com has been set up for programmers. Eventually all NTER source code will go in repositories on this page.

An Organization on GitHub is a place where all of the programming project code repositories for a group can be located. See ours at: https://github.com/NTERCom

To get permission to upload code, Email the administrator and ask to be assigned to the appropriate team: Administrator Mail

The NTER Com organization on GitHub has two repositories:

  • LMS

    This is for the NTER LMS code (aka Ilias/NWTP). This is the code that enables course authoring and configuration.

    LMS Repository Link: https://github.com/NTERCom/LMS

  • Portal

    This is for the LifeRay server setup and Portlets that run under LifeRay. This generates the screens that the students see when they first log in.

    Portal Repository Link" https://github.com/NTERCom/Portal
